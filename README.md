# Sprachhilfe für macOS

Sprachhilfe ist eine native macOS-App für Spracheingabe, Workflows und KI-gestützte Textverarbeitung. Dieses Repository enthält die offiziellen Downloads und Update-Informationen der App.

## Download

[Neueste Version herunterladen](https://github.com/tamaga3131400/sprachhilfe-dist/releases/latest)

- Erfordert macOS 14 oder neuer.
- DMG öffnen und Sprachhilfe in den Ordner „Programme“ ziehen.
- Die offizielle DMG ist Developer-ID-signiert und von Apple notarisiert.

## Was Sprachhilfe bietet

- **Lokale KI auf Apple Silicon:** Quantisierte MLX-Modelle laufen direkt auf dem Mac, ohne separaten Server.
- **Eigene Workflows:** System-Prompts und Workflows lassen sich speichern und gezielt für wiederkehrende Aufgaben einsetzen.
- **Eigene Wissensquellen:** Dokumente werden nur bewusst für den jeweiligen Kontext freigegeben.

## Updates

Sprachhilfe erkennt neue Versionen über die integrierte Update-Funktion. Du kannst in der App jederzeit manuell nach Updates suchen.
